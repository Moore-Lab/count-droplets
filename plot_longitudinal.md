# plot_longitudinal.py

Interactive overview plot for a longitudinal analysis output file.  Loads a `longitudinal_analysis_*.npy` file produced by `longitudinal_analysis.py` and renders a single multi-panel figure showing all extracted parameters vs chunk mid-time, with ±1 std dev shaded bands.

## Usage

Edit `LONGITUDINAL_FILE` in the `__main__` block to point to your file, then run:

```bash
python plot_longitudinal.py
```

If the exact filename is not found, the script automatically selects the **most recent** `longitudinal_analysis_*.npy` in the same directory.

To save the figure instead of displaying it, set `SAVE_PATH`:

```python
SAVE_PATH = "plots/my_video/longitudinal_overview.png"
```

## Figure layout

The figure has 5 rows and 2 columns:

| Row | Left panel | Right panel |
|-----|-----------|-------------|
| 1 | Count/frame ± std (blue), density twin axis (navy, droplets/mm³) | Bar chart: total droplets per chunk |
| 2 | Streak width ± std in pixels (orange), twin axis in µm (brown) | Total speed ± std (mm/s) |
| 3 | Radial velocity \|v_r\| ± std (mm/s) | Axial velocity \|v_z\| ± std (mm/s) |
| 4 | Wind direction — circular mean angle (°), scatter colored by time | Anisotropy ratio scatter, dashed line at 1 (isotropic) |
| 5 (full width) | Speed component summary: \|v\|, \|v_r\|, \|v_z\| on one axes | — |

Shaded bands show ±1 standard deviation. Scatter plots in row 4 are colored by chunk mid-time using the viridis colormap.

## Prerequisites

A `longitudinal_analysis_*.npy` file generated by `longitudinal_analysis.py`.  No other scripts need to be running.

## Settings

| Variable | Description |
|----------|-------------|
| `LONGITUDINAL_FILE` | Full path to the `.npy` file to plot |
| `SAVE_PATH` | Output image path, or `None` to display interactively |
